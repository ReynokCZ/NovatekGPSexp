# NovatekGPSexp v1 - WINDOWS BATCH SCRIPT
# version 1 - date 24.04.2019
# Autor: Reynok Flegmatyk, email: reynok@greenmail.net

It is single file Batch script for generating .gpx datafiles with GPS coordinates from all videos in folder at once.

Insert the NovatekGPSexp.bat file along with the nvtk_mp42gpx.py script into the video files folder and run it.

You must have installed Python language interface to running python scripts.
https://www.python.org/
Tested with Python version 3.7 and you must check "Add Python 3.7 to PATH" for right work batch.

Reference: Sergei Franco python script nvtk_mp42gpx.py "https://sergei.nz/extracting-gps-data-from-viofo-a119-and-other-novatek-powered-cameras/"
